Get started

The app is available on the App Store. You can download and install it using one of the following options:

  • Scan the QR code with your camera and follow the link that appears on the screen
  • Click the button to open the app directly on the App Store

Learn Basics

The app works in a way that significantly reduces time spent on routine tasks, allowing you to focus solely on adjustments and individual tasks. Layout modes such as Auto, Grid, and Collage arrange photos on pages automatically, providing you with tools to adjust layout behaviour and facilitate subsequent editing.

For example, you can add 100 photos, and the app will automatically place them on multiple pages in just a few seconds, letting you print all of them right away.

The app allows you to create an unlimited number of projects with desired settings and content. Each project supports both standard and custom paper (page) sizes, but only one size can be selected for a single project at a time.

Each page within the project can have individual orientation (Landscape or Portrait), margins, spacing, background color, and background images, all of which can be changed at any moment.

To prevent any page from being updated, use the "Lock" button located just below the current page. You will see multiple indications throughout the app that the page is locked. To unlock a page, toggle the "Lock" button below the page or use the page popup menu that appears over each page on the pages bar.

The app provides tools for both editing photos on a page and individual photo editing. To edit photos on any page, you need to select them first. The app supports both single-target selection and multi-select.

To edit any photo individually, use a double tap gesture or the Edit Photo action from the selection action list.

To export your results use the Print or the Share button.

Add photos to project

You can add as many photos as you wish to the project multiple times. However, we do not recommend selecting more than 100 photos in one go.

Add photos to the current page

Ensure that the current page is unlocked before attempting to add photos to it.

  1. Open the main screen.
  2. Tap the Add button on the bottom bar.
  3. Choose source: Files, Camera, or Photo Library.
  4. Select photos from the source.
  5. After a moment they will appear in the project.

Set page size, margins and spacing

You can change these settings at any time. The new size will be applied to all pages in the project, while margins and spacing will apply to the current page, or all pages when the "Apply to All" option is selected.

Select or add paper size

To change the page size, select one of the standard sizes or add a new one:

  1. Go to the main screen.
  2. Tap the second button on the bottom bar.
  3. Choose desired paper size from the list.
  4. Tap on any text field below the list to enter and apply a new size.

Specify page orientation, margins and spacing

  1. Tap the Page Settings button on the page bar.
  2. Choose either Portrait or Landscape for the page orientation.
  3. Adjust the margins manually using the circle markers or enter new values in the four text fields on the page.
  4. Enable the "Equal margins" option to apply the same margins to each side.
  5. Specify a new value for page spacing using the text field below the page.
  6. Enable the "Apply to All" option to apply the page settings to all non-locked pages.

Choose layout mode

There are four layout modes in the app to choose from: Normal, Auto, Grid, and Collage. Each mode offers both standard and unique tools and edits to achieve different results. You can select individual layout for each page or use one layout mode for all pages in the project. Note that a small icon indicating the selected layout mode also appears near the current page scale.

Select layout for the current page

You can change the layout mode at any time, even when the page is full of content.

  1. Go to the main screen.
  2. Tap the Page Layout button on the page bar.
  3. Select one of the four possible layouts that appear above the page.
  4. Specify the number of columns for the Grid layout mode.
  5. Choose a template or create a new table for the Collage layout mode.

Specify layout for all pages

For all modes, except Normal, this also organizes photos in a continuous flow, removing blank spaces and deleting empty pages. Removing photos at the beginning will move all subsequent photos backward to fill the gaps.

  1. Tap the "Apply to All" button for Normal mode.
  2. Tap the "Apply to All and Auto Arrange" for other modes.

Set resolution

Resolution is used for both calculating pixel size of photos and pages during editing, as well as calculating pixel size for images during export.

Select or add resolution

Resolution defines the density of pixels per unit of measurement. You can choose between DPI (dots per inch) and DPCM (dots per centimeter) resolution units. To change project resolution:

  1. Go to the main screen.
  2. Tap the right button on the bottom bar.
  3. Choose your preferred resolution from the list.
  4. Tap on the text field above the list to enter and apply a new resolution.
  5. Use buttons at the bottom to switch between DPI and DPCM.

Edit page content

The app provides a comprehensive set of tools to move, resize, orient, duplicate, and remove photos. To learn more about how to edit collage pages, visit the Collage Editing section.

Edit page content

  1. Tap or long press any photo to select it.
  2. Long-press and move from any unoccupied point on the page to multi-select.
  3. Resize the selected photos using markers.
  4. Move the selected photos using a pan gesture.
  5. Tap the Copy button to make a duplicate of the selected photos.
  6. Tap the Rotate button to rotate the selected photos clockwise by 90°.
  7. Tap the "N Selected" button in the middle of the selection bar to open the editing popup list.
  8. Use buttons from the editing popup list to edit, crop, flip, delete, bring to front, or send to back the selected photos.
  9. Tap the Make 4 Copies button from the editing popup list to make 4 copies at once when one photo is selected.

Update selected photos size and position

  1. Tap the "Photo Size" button on the bottom bar.
  2. Enter the Left and Top positions using the keyboard.
  3. Enter the Width (W) and Height (H) using the keyboard.
  4. Choose a new size from the list.
  5. Select either "Original", "Proportional", or "Square" sizes from the list.

Update size of all photos

  1. Tap on the drop-down menu button at the top of the photo size popup.
  2. Select either "All photos" or "Page photos".
  3. Make any size changes described in the previous section to resize either all photos in the project or all photos on the page.
  4. Select the Smart mode on the bottom bar to apply a new size to photos based on their Portrait or Landscape orientation.

Add border to photos

You can add a colored border to selected photos, all photos on a page, or all photos in the project.

  1. Go to the main screen.
  2. Tap the Border button on the bottom bar.
  3. Select the border position: Inside, Center, or Outside.
  4. Select a color from the palette.
  5. Tap again the selected color to pick/change its RGB value.
  6. Long press any color in the palette to pick/change its value, or reset to default.
  7. Adjust the border thickness using the slider.
  8. Enter the border thickness into the text field using the keyboard.
  9. Tap on the drop-down menu button at the top of the border popup to change editing scope to "All photos" or "Pages photos".

Edit photo individually

You can edit any single photo by double-tapping it on the page or selecting it and then choosing the Edit Photo option from the editing popup window.

Edit selected photo

  1. Use standard pinch and pan gestures to scale and position the photo inside the crop frame.
  2. Move the bottom slider to rotate the photo in the range of 0 - 360 degrees.
  3. Tap the Flip button on the top bar to flip the photo horizontally.
  4. Tap the Flip button on the top bar to flip the photo vertically.
  5. Use the menu on the top bar to reset the photo to its initial state.

Complete photo editing

  1. Tap the 'Done' button on the top bar to apply all the editing changes.
  2. Tap the 'Cancel' button on the top bar to cancel all the editing changes.

Create stunning collages

The app allows you to use Collage layout mode to create beautiful compositions from your photos. When this mode is active, the app editing interface receives some minor updates. To learn more about page layout modes, visit the Choose layout mode section.

Select collage template

  1. Open the main screen.
  2. Tap the Page Layout button on the page bar.
  3. Tap the "Collage" layout mode above the page.
  4. Select either the Template or Table collage type on the bottom bar of the popup window.
  5. Choose the most appropriate template from the list or create a table of the preferred size.

Edit current collage page

Blank collage cells, called "placeholders", are an essential and irremovable part of the collage layout structure. They can also be the target of some editing operations.

  1. Tap or long press any photo/placeholder to select it.
  2. Move the selected photo/placeholder onto another photo/placeholder to swap them.
  3. Move marker on the edge of each photo/placeholder to resize collage segment.
  4. Move any marker to the edge of another segment to merge photos/placeholders.
  5. Tap the Split button to divide the selected photo/placeholder into two equal vertical parts.
  6. Tap the Split button to divide the selected photo/placeholder into two equal horizontal parts.
  7. Tap the Rotate button to rotate the selected photo clockwise by 90°.
  8. Tap the Unbind button to detach selected collage cell from others during editing.
  9. Tap the "N Selected" button in the middle of the selection bar to open the editing popup list.
  10. Use buttons from the editing popup list to edit, crop, flip, delete the selected photo.

Update size and position of selected photo or placeholder.

This editing operation for Collage is very similar to the one for the Normal page, described in this section.

Print and Share your work

You can print your document using standard printing menu or share it in PDF, JPEG, and TIFF formats. To print directly from the app your printer should support AirPrint technology. The app uses the "Best" image quality when preparing the project for printing.

Print your work

  1. Open the main screen.
  2. Tap the Print button on the top bar.
  3. Make sure all required settings are properly configured on that standard screen, including paper size, quality, color, and other presets.
  4. Tap the "Print" button in the top right corner of the popup window.

Share as PDF

  1. Go to the main screen.
  2. Tap the Share button on the top bar.
  3. Enter a filename for your PDF document.
  4. Choose the image quality between Poor, Good, High, and Best.
  5. Tap the "Export" button at the bottom of the popup.
  6. Choose a destination to share.

Share as JPEG, TIFF, PNG, HEIC

  1. Go to the main screen.
  2. Tap the Share button on the top bar.
  3. Tap the drop-down menu button at the top of the popup and choose either JPEG, TIFF, PNG or HEIC format.
  4. Enter a filename for your image files. This name will also be used to create an album when saving to the Photo Library.
  5. Specify the compression for JPEG and HEIC using the slider or the text field.
  6. Tap the "Export" button at the bottom of the popup.
  7. Choose a destination to share.
